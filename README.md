## 🧠 General Problem-Solving Strategies

Four strategies that work across *every* unit above, worth trying whenever you're stuck (from AOPS Prealgebra):

1. **Find a Pattern** — compute the first several small cases, spot a repeating cycle or rule, then extrapolate to the actual number.
2. **Make a List** — organize all possibilities systematically (most restrictive condition first) so nothing is missed or double-counted.
3. **Draw a Picture** — sketch the situation (distances, timelines, a graph of connections) to make hidden structure visible before calculating.
4. **Work Backwards** — when a problem describes a chain of steps ending at a known result, undo each step in reverse order to recover the starting value (see [[13_Logic_and_Magic_Squares]] for worked examples).
